# project-IC-frontend

## Overview
Project IC is an interview preparation platform designed for users including developers, students, and job aspirants. It provides a comprehensive system with three user roles: user, admin, and superadmin. Users can access interview questions sourced from coaching institutes, along with additional resources to aid in their interview preparation.

## Features
- Interview questions
- Latest technology insights
- FAQs
- User authentication
- PDF notes
- And more

## Tech Stack
- **Frontend**: React
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Routing**: React Router
- **HTTP Client**: Axios

## Installation
To install the project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/Smruti-Ranjan-Sahoo-Tech/project-IC-frontend.git
   ```
2. Navigate to the project directory:
   ```bash
   cd project-IC-frontend
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage
To start the development server, run:
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:3000` to view the application.

## Architecture
The project is structured into several key directories:
- **src**: Contains the main application code including components, pages, and routing logic.
- **public**: Holds static assets such as images and icons.
- **API**: Contains the Axios instance for making HTTP requests.
